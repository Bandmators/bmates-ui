# bmates-ui

<br />

## Install

```
npm i bmates-ui
```

<br />

## How to use?

<br />

### BMatesProvider

Add Styled Component, `BMatesProvider`

```
import { BMatesProvider } from 'bmates-ui';

<BMatesProvider>
    <!-- ... -->
</BMatesProvider>
```

<br />
